## ✨ Features

- ✅ **6 conversion categories** — Temperature, Distance, Mass, Speed, Area, Time
- ✅ **Bidirectional inputs** — type in either box, the other auto-calculates
- ✅ **Dark and Light mode** toggle
- ✅ **Swap button** — instantly flip the conversion direction
- ✅ **Copy button** — copy result to clipboard
- ✅ **Formula display** — shows the math formula used
- ✅ **Keyboard shortcuts** — convert with Enter, clear with Esc
- ✅ **C program** — full terminal-based converter with colored output

## 📐 All Conversion Formulas

### 🌡️ Temperature

| Conversion | Formula |
|---|---|
| Celsius → Fahrenheit | `°F = (°C × 9/5) + 32` |
| Fahrenheit → Celsius | `°C = (°F − 32) × 5/9` |
| Celsius → Kelvin | `K = °C + 273.15` |
| Kelvin → Celsius | `°C = K − 273.15` |
| Fahrenheit → Kelvin | `K = (°F − 32) × 5/9 + 273.15` |

### 📏 Distance

| Conversion | Formula |
|---|---|
| Meter → Kilometer | `km = m ÷ 1000` |
| Kilometer → Meter | `m = km × 1000` |
| Meter → Centimeter | `cm = m × 100` |
| Meter → Feet | `ft = m × 3.28084` |
| Kilometer → Mile | `mi = km × 0.621371` |
| Centimeter → Inch | `in = cm ÷ 2.54` |

### ⚖️ Mass

| Conversion | Formula |
|---|---|
| Kilogram → Gram | `g = kg × 1000` |
| Gram → Kilogram | `kg = g ÷ 1000` |
| Kilogram → Pound | `lb = kg × 2.20462` |
| Pound → Kilogram | `kg = lb ÷ 2.20462` |
| Gram → Milligram | `mg = g × 1000` |
| Kilogram → Metric Ton | `t = kg ÷ 1000` |

### 💨 Speed

| Conversion | Formula |
|---|---|
| m/s → km/h | `km/h = m/s × 3.6` |
| km/h → m/s | `m/s = km/h ÷ 3.6` |
| km/h → mph | `mph = km/h × 0.621371` |
| mph → km/h | `km/h = mph ÷ 0.621371` |
| m/s → mph | `mph = m/s × 2.23694` |

### 📐 Area

| Conversion | Formula |
|---|---|
| m² → cm² | `cm² = m² × 10,000` |
| m² → ft² | `ft² = m² × 10.7639` |
| km² → mi² | `mi² = km² × 0.386102` |
| Hectare → m² | `m² = ha × 10,000` |
| Acre → m² | `m² = acre × 4046.86` |

### ⏱️ Time

| Conversion | Formula |
|---|---|
| Second → Minute | `min = s ÷ 60` |
| Minute → Hour | `h = min ÷ 60` |
| Hour → Day | `day = h ÷ 24` |
| Second → Hour | `h = s ÷ 3600` |
| Day → Week | `week = day ÷ 7` |
| Second → Millisecond | `ms = s × 1000` |
