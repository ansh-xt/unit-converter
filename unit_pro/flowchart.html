How to Connect HTML Frontend to C Backend

C is a compiled language — it runs as an executable, not directly in a browser. Here are the three ways to bridge them, from easiest to most advanced.

METHOD HOW IT WORKS DIFFICULTY BEST FOR
📁 File Handling JS writes input to input.txt → C reads it → C writes result to output.txt → JS reads output.txt ⭐ Beginner Local projects, Class 11
🌐 CGI Script Apache/Nginx web server runs the compiled C program as a CGI script and returns HTML ⭐⭐ Medium Old-school web apps
🔌 REST API (Python) Python Flask/FastAPI calls the C program via subprocess, exposes a JSON API that JS fetches ⭐⭐⭐ Advanced Production web apps

File Handling Method (Beginner Code Example)

C side — reads input.txt, converts, writes output.txt
JS side — writes input.txt using File API, reads output.txt

// C: read input → convert → write output
FILE *fin = fopen("input.txt", "r");
fscanf(fin, "%lf", &value);
fclose(fin);

result = (value * 9.0/5.0) + 32.0;

FILE *fout = fopen("output.txt", "w");
fprintf(fout, "%.2f", result);
fclose(fout);
